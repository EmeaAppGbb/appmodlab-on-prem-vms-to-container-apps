## рҹҢҲ OVERVIEW

**Welcome to the future, time traveler!** рҹ•№пёҸ

Remember when deploying apps meant:
- рҹҗҢ Waiting for VMs to boot (grab a coffee... or two)
- рҹ”§ Manual patching every Tuesday at 3 AM
- рҹ’ё Paying for VMs that sit idle 23 hours a day
- рҹ”Ҙ Hardcoded IPs everywhere like it's 1999

**LEVEL UP!** This lab transforms a crusty three-VM veterinary clinic system into a sleek, auto-scaling, container-powered masterpiece running on **Azure Container Apps** with **Dapr** magic! рҹҺҜ

### рҹҸҘ Meet PawsCare Veterinary Network

**Legacy System (aka "The Before Times"):**
- рҹ–ҘпёҸ **VM 1:** Windows Server 2019 + IIS + ASP.NET Core (The Web Frontend)
- рҹ–ҘпёҸ **VM 2:** Ubuntu + Node.js + Express + MongoDB (The API Server)
- рҹ–ҘпёҸ **VM 3:** Ubuntu + Python + Celery + RabbitMQ (The Background Worker)
- рҹ“Ғ SMB file shares (because it's 2015 forever)
- рҹ”— Hardcoded IPs (10.0.1.10, anyone?)
- рҹҳӯ Manual everything

**Target System (aka "The Glow-Up"):**
- рҹ“Ұ **Container 1:** ASP.NET Core in a Docker container
- рҹ“Ұ **Container 2:** Node.js API in a Docker container
- рҹ“Ұ **Container 3:** Python worker in a Docker container
- вҳҒпёҸ **Azure Container Apps** (serverless magic)
- рҹҺӯ **Dapr** for service mesh awesomeness
- рҹ“Ҡ **Azure Cosmos DB** + **Azure SQL**
- рҹ“Ё **Azure Service Bus** (bye RabbitMQ!)
- рҹ—ӮпёҸ **Azure Blob Storage** (bye SMB!)
- вҡЎ **KEDA auto-scaling** (scale to zero, scale to hero!)

## рҹ–ҘпёҸ LEGACY STACK (The Old World)

### **Three VMs Walk Into a Bar...**

**VM 1: The Web Frontend** рҹӘҹ
- Windows Server 2019
- IIS 10 hosting ASP.NET Core 3.1 MVC
- SQL Server 2019 (shared with the app)
- 4 vCPU, 8GB RAM (always on, always billing)
- IP: `10.0.1.10`

**VM 2: The API Server** рҹҗ§
- Ubuntu 20.04
- Node.js 14.x + Express 4.x
- MongoDB 4.4 for pet records
- 2 vCPU, 4GB RAM
- IP: `10.0.1.20`

**VM 3: The Background Worker** рҹ”§
- Ubuntu 20.04
- Python 3.8 + Celery + RabbitMQ
- Cron jobs for appointment reminders
- 2 vCPU, 4GB RAM
- IP: `10.0.1.30`

### **Anti-Patterns Spotted!** рҹҡЁ

- вқҢ Hardcoded IPs everywhere
- вқҢ No auto-scaling (overpay for idle resources)
- вқҢ Manual patching nightmares
- вқҢ SMB file shares (it's not 2010!)
- вқҢ Single-instance RabbitMQ (no HA)
- вқҢ Manual SSL certificate deployment
- вқҢ No health checks
- вқҢ Monolithic VM updates
